*Test Case #1: QA0 edit the final comment*

1. Go to Manual Search tab
2. Go to ExecBlocks (QA0) tab
3. In Interval select Other
4. In From select some date of 2014//01/01
5. In To select the current date
6. Click in Search button
7. Select the first result
8. Click the Edit icon of Final QA0 Comment
9. Input some Text and Save
10. Expected Result: The Final QA0 comment should be showed in the panel
11. Passed: OK (testcaste01.png ICT-3500-TestCase01.html).
